Welcome to the nghttp2 priority demo

In order to run this demo, you need to:

  • Download and install nghttp2
  • Create a folder for the demo app, for instance 'http2'
  • Place the entire docs project under it in a folder named 'docs'
  • Create a self-signed certificate, for instance using 'openssl' and place it in the folder 'http2/certs'
  • Run nghttp2 using the command 'nghttpd 443 certs/ca.key certs/ca.crt -d docs' from the 'http2' folder
Note that you need administrator privileges (or root) to open port 443. If you don't have permissions, you can use another port
